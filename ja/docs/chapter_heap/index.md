# 第 8 章 &nbsp; ヒープ

![ヒープ](../assets/covers/chapter_heap.jpg){ class="cover-image" }

!!! abstract

    ヒープは山とその険しい峰のように、層をなして起伏し、それぞれが独特の形を持っています。

    各山の頂は散らばった高さで上下しますが、最も高いものが常に最初に注目を集めます。
