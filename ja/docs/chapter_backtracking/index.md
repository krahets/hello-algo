# 第 13 章 &nbsp; バックトラッキング

![バックトラッキング](../assets/covers/chapter_backtracking.jpg){ class="cover-image" }

!!! abstract

    迷路の探検家のように、私たちは前進する道で障害に遭遇することがあります。

    バックトラッキングの力は、私たちに新しく始めること、試し続けること、そして最終的に光への出口を見つけることを可能にします。
