# 第 5 章 &nbsp; スタックとキュー

![スタックとキュー](../assets/covers/chapter_stack_and_queue.jpg){ class="cover-image" }

!!! abstract

    スタックは積み重ねられた猫のようなもので、キューは一列に並んだ猫のようなものです。

    それらはそれぞれ、後入先出（LIFO）と先入先出（FIFO）の論理関係を表しています。
